## Google Meet ŌĆö Keyboard Shortcuts

Google Meet is a video conferencing web app accessed at meet.google.com. The agent interacts with it in a browser window.

> **Prerequisite:** To see a complete list of shortcuts while in a meeting, press `Shift + ?`.

### Meeting Controls

| Action | Windows / ChromeOS | Mac |
|---|---|---|
| Mute or unmute microphone | `Ctrl + d` | `Ōīś + d` |
| Turn camera on or off | `Ctrl + e` | `Ōīś + e` |
| Start or stop presenting | `Ctrl + Alt + t` | `Ctrl + Ōīś + t` |
| Show or hide captions | `c` or `Shift + c` | `c` or `Shift + c` |
| Turn on captions (while presenting) | `Ctrl + Shift + c` | `Ctrl + Shift + c` |
| Open picture-in-picture mode | `Shift + m` | `Shift + m` |
| Push to Talk (must enable in settings) | Hold `Spacebar` | Hold `Spacebar` |

### Participant & Layout Controls

| Action | Windows / ChromeOS | Mac |
|---|---|---|
| Show or hide participants panel | `Ctrl + Alt + p` | `Ctrl + Ōīś + p` |
| Show or hide meeting chat window | `Ctrl + Alt + c` | `Ctrl + Ōīś + c` |
| Raise or lower your hand | `Ctrl + Alt + h` | `Ctrl + Ōīś + h` |
| Increase number of participant tiles | `Ctrl + Alt + k` | `Ctrl + Ōīś + k` |
| Decrease number of participant tiles | `Ctrl + Alt + j` | `Ctrl + Ōīś + j` |
| Minimize or expand your self-view video | `Ctrl + Alt + m` | `Ctrl + Ōīś + m` |

### Accessibility & Announcements

| Action | Windows / ChromeOS | Mac |
|---|---|---|
| Announce who is currently speaking | `Ctrl + Alt + s` | `Ctrl + Ōīś + s` |
| Announce current room information | `Ctrl + Alt + i` | `Ctrl + Ōīś + i` |
| Change how you hear reactions | `Shift + r` | `Shift + r` |
| Show keyboard shortcuts help | `Shift + ?` | `Shift + ?` |

### Tips

- **Mute etiquette:** Use `Ctrl + d` to quickly mute/unmute before speaking.
- **Push to Talk:** Must be enabled in Meet Settings first. Hold `Spacebar` to temporarily unmute while muted.
- **Presenting:** `Ctrl + Alt + t` opens the "Present now" dialog ŌĆö agent must then select a tab, window, or entire screen.
- If the chat or participants panel is open, close it with the same shortcut (toggle).
