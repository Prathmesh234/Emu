## Web Search

You can search the web for the user by opening their default browser.

### How to search

1. Open the browser using keyboard shortcut or app launcher
2. Navigate to the address bar (Ctrl+L on Windows, Ctrl+L on Linux)
3. Type the search query or URL
4. Press Enter and wait for results to load
5. Take a screenshot to read the results

### Tips

- For quick factual searches, use the address bar directly — most browsers
  default to Google/DuckDuckGo search from the address bar.
- If the user says "search for X", type the query in the address bar.
- If the user gives a specific URL, type it directly.
- Wait 2-3 seconds after pressing Enter for the page to load before
  taking a screenshot.
- If results are below the fold, scroll down and take another screenshot.
- For research tasks, open multiple tabs (Ctrl+T) to compare sources.

### Search strategies

- **Quick answer**: Search → read top result → report back
- **Research**: Search → open 2-3 results in tabs → synthesize
- **Specific site**: Go directly to the URL the user provides
- **Documentation**: Search "[topic] docs" or "[topic] documentation"
