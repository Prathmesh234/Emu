## Microsoft Excel — Keyboard Shortcuts (macOS)

Microsoft Excel is a spreadsheet application. On macOS it runs as a native desktop app. The interface has a ribbon toolbar at the top (Home, Insert, Draw, Formulas, Data, Review, View tabs). Below the ribbon is the formula bar showing the active cell reference and its contents. The main area is the cell grid. Sheet tabs at the bottom allow switching between worksheets. The status bar shows averages, counts, and sums of selected cells.

### Tips

- **Formula entry:** Press `=` to start a formula, then click cells to add references. Press `⌘ + T` or `F4` to cycle between absolute (`$A$1`), relative (`A1`), and mixed references.
- **Quick formatting:** `Control + Shift + $` for currency, `Control + Shift + %` for percentage — no need to open the format dialog.
- **Sheet navigation:** `Option + ←/→` switches between sheet tabs instantly.
- **AutoSum:** Select a cell below a column of numbers, press `⌘ + Shift + T` to auto-insert a SUM formula.
- **Flash Fill:** `Control + E` can auto-detect patterns and fill data.
- Press `⌘ + Option + R` to collapse the ribbon for more spreadsheet space.
