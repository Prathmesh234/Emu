## Microsoft Word ŌĆö Keyboard Shortcuts (macOS)

Microsoft Word is a document editor. On macOS it runs as a native desktop application. The interface has a ribbon toolbar at the top with tab groups (Home, Insert, Design, Layout, References, Mailings, Review, View). The main canvas is the document editing area. An optional Navigation Pane on the left helps jump between headings, pages, or search results. The status bar at the bottom shows word count, page number, and view options.

### Tips

- **Heading styles** (`Ōīś + Option + 1/2/3`) make documents navigable ŌĆö the Navigation Pane (`View > Navigation Pane`) lists all headings for quick jumping.
- **Format Painter:** Copy formatting with `Ōīś + Shift + C`, click the target text, then paste with `Ōīś + Shift + V`.
- **Track Changes:** Toggle with `Ōīś + Shift + E` ŌĆö useful for collaborative editing.
- **Quick navigation:** `Ōīś + F` opens the search sidebar; `Ōīś + Shift + H` opens Find & Replace.
- Press `Ōīś + Option + R` to collapse the ribbon and gain more editing space.
