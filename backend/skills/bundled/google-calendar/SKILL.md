## Google Calendar — Keyboard Shortcuts

Google Calendar is accessed at calendar.google.com. Keyboard shortcuts must be turned on in Settings.

### How to Enable Keyboard Shortcuts

1. Open Google Calendar → click the **gear icon** (top-right) → **Settings**
2. Check **Enable keyboard shortcuts**
3. Click **Save**

> Press `?` (Shift + /) inside Calendar to see the shortcut help overlay.

---

### Navigation — Move Around the Calendar

| Action | Shortcut |
|---|---|
| Previous time period (day/week/month) | `p` or `k` |
| Next time period (day/week/month) | `n` or `j` |
| Go to today | `t` |
| Go to a specific date | `g` (opens date picker) |

### Change Your Calendar View

| Action | Shortcut |
|---|---|
| Day view | `1` or `d` |
| Week view | `2` or `w` |
| Month view | `3` or `m` |
| Custom view (X days) | `4` or `x` |
| Schedule / list view | `5` or `a` |
| Year view | `6` or `y` |

### Create & Edit Events

| Action | Shortcut |
|---|---|
| Create new event | `c` |
| View event details | `e` (with event selected) |
| Delete event | `Delete` or `Backspace` |
| Undo last action | `z` |
| Save event (in event editor) | `Ctrl + s` |
| Back to calendar grid | `Esc` |

### Quick Actions

| Action | Shortcut |
|---|---|
| Search | `/` |
| Open Settings | `s` |
| Show keyboard shortcuts help | `?` (Shift + /) |
| Go to side panel (Tasks, Keep, etc.) | `Alt + Shift + .` |
| Return from side panel | `Alt + Shift + ,` |
| Refresh calendar | `r` |

### View Tasks & Keep

| Action | Shortcut |
|---|---|
| Open Tasks side panel | Click Tasks icon or `Alt + Shift + .` then navigate |
| Open Google Keep side panel | Click Keep icon or `Alt + Shift + .` then navigate |

### Tips

- **Single-key shortcuts:** Most Calendar shortcuts are single-key presses (no Ctrl needed).
- **Date navigation:** `p`/`n` move backward/forward by whatever view you're in (e.g., one week in week view, one month in month view).
- **Quick event creation:** Press `c` to open the event creation dialog, type details, and press `Ctrl + s` to save.
- **Go to date:** Press `g`, type a date (e.g., "12/25/2025"), and press Enter.
- To switch views quickly, just press the number keys `1`–`6`.
