## Microsoft OneNote ‚Äî Keyboard Shortcuts (macOS)

Microsoft OneNote is a digital notebook application. On macOS it runs as a native desktop app. The interface shows a notebook list in the left panel, section tabs at the top or left (depending on layout), a page list on the side, and a free-form canvas in the center. Content blocks (text, images, drawings) can be placed anywhere on the canvas. Unlike Word, OneNote pages have unlimited height and content is organized in moveable containers.

### Prerequisites

Enable **Full Keyboard Access** for the best keyboard experience:

1. Open **System Settings** ‚Üí **Accessibility**
2. Select the **Navigation** tab
3. Enable **Full Keyboard Access**

### Tips

- **Free-form canvas:** Unlike Word, OneNote lets you click anywhere on a page to start typing ‚Äî content lives in moveable containers.
- **Outline collapsing:** Use `Control + Shift + +/-` to collapse/expand indented content under a heading.
- **Section switching:** `‚åò + Shift + {/}` cycles between sections in the current notebook.
- **Page management:** `‚åò + Shift + M` moves a page to a different section or notebook ‚Äî useful for reorganizing.
- **Quick date stamp:** `‚åò + D` inserts today's date; `‚åò + Shift + D` inserts date and time.
- **Tables:** Press `Tab` in normal text to auto-create a table. Keep pressing `Tab` to add columns.
