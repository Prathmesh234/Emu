## Google Docs — Keyboard Shortcuts

Google Docs is a web-based word processor at docs.google.com.

> Press `Ctrl + /` (Windows) or `⌘ + /` (macOS) to open the keyboard shortcuts help dialog inside Docs.
> Press `Alt + /` (Windows) or `Option + /` (macOS) to open Tool finder (search the menus).

### Tips

- **Two-step shortcuts:** Sequences like "Hold `Ctrl + Alt`, press `n` then `c`" mean: hold Ctrl+Alt (or ⌘+Option on Mac), tap `n`, release all, then tap `c`.
- Use `Alt + /` (Windows) or `Option + /` (macOS) — Tool finder — to quickly search for any menu command by name.
- `Ctrl + Alt + m` / `⌘ + Option + m` is the fastest way to add a comment on selected text.
- Heading styles (`Ctrl + Alt + 1-6` / `⌘ + Option + 1-6`) are essential for document structure and the outline sidebar.
- On macOS, `⌘` replaces `Ctrl`, `Option` replaces `Alt`, and for Google Docs menus use `Ctrl + Option + letter`.

### Tips

- **Two-step shortcuts:** Sequences like "Hold `Ctrl + Alt`, press `n` then `c`" mean: hold Ctrl+Alt, tap `n`, release all, then tap `c`.
- Use `Alt + /` — Tool finder — to quickly search for any menu command by name.
- `Ctrl + Alt + m` is the fastest way to add a comment on selected text.
- Heading styles (`Ctrl + Alt + 1-6`) are essential for document structure and the outline sidebar.
