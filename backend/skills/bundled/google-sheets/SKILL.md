## Google Sheets — Keyboard Shortcuts

Google Sheets is a web-based spreadsheet app at sheets.google.com.

> Press `Ctrl + /` to open the keyboard shortcuts help dialog inside Sheets.
> Press `Alt + /` to open Tool finder (search the menus).
> To enable compatible spreadsheet shortcuts: Help → Keyboard shortcuts → toggle **Enable compatible spreadsheet shortcuts**.

### Tips

- **F2** is essential — press it to enter edit mode in a cell, or to toggle between edit mode and reference-selection mode in a formula.
- **F4** cycles through absolute/relative reference modes (`A1` → `$A$1` → `A$1` → `$A1`).
- **Ctrl + ~** toggles formula view — see all formulas in the sheet at once.
- Use `Ctrl + Shift + v` to paste values only (strip formatting and formulas).
- **Fill down** (`Ctrl + d`) and **Fill right** (`Ctrl + r`) are faster than copy-paste for repeating values/formulas.
- Press `Ctrl + a` once to select the data region; press again to select the entire sheet.
