## Google Sheets — Keyboard Shortcuts

Google Sheets is a web-based spreadsheet app at sheets.google.com.

> Press `Ctrl + /` (Windows) or `⌘ + /` (macOS) to open the keyboard shortcuts help dialog inside Sheets.
> Press `Alt + /` (Windows) or `Option + /` (macOS) to open Tool finder (search the menus).
> To enable compatible spreadsheet shortcuts: Help → Keyboard shortcuts → toggle **Enable compatible spreadsheet shortcuts**.

### Tips

- **F2** is essential — press it to enter edit mode in a cell, or to toggle between edit mode and reference-selection mode in a formula.
- **F4** cycles through absolute/relative reference modes (`A1` → `$A$1` → `A$1` → `$A1`). On macOS, use `Fn + F4` if function keys are set to system functions.
- **Ctrl + ~** / **Ctrl + ~** toggles formula view — see all formulas in the sheet at once.
- Use `Ctrl + Shift + v` / `⌘ + Shift + v` to paste values only (strip formatting and formulas).
- **Fill down** (`Ctrl + d` / `⌘ + d`) and **Fill right** (`Ctrl + r` / `⌘ + r`) are faster than copy-paste for repeating values/formulas.
- Press `Ctrl + a` / `⌘ + a` once to select the data region; press again to select the entire sheet.
- On macOS, `⌘` replaces `Ctrl`, `Option` replaces `Alt`, and for Google Sheets menus use `Ctrl + Option + letter`.
