## Google Drive — Keyboard Shortcuts

Google Drive is the file management web app at drive.google.com.

> Press `Ctrl + /` to display the keyboard shortcuts list inside Drive.

---

### Navigation and Views

| Action | Windows/ChromeOS |
|---|---|
| Go to navigation panel (folders list) | `Alt + g` then `n` or `Alt + g` then `f` |
| Go to items view | `Alt + g` then `i` |
| Switch between grid and list in items view | `Alt + v` then `l` |
| Go to page content | `Alt + g` then `c` |
| Go to details pane | `Alt + g` then `d` |
| Go to top of application (Google bar) | `Alt + g` then `t` |
| Go to download status | `Alt + g` then `a` |
| Go to upload status | `Alt + g` then `u` |
| Show or hide approvals pane | `Alt + v` then `e` |
| Show or hide details pane | `Alt + v` then `d` |
| Show or hide activity pane | `Alt + v` then `a` |

### Tips

- **Two-step shortcuts:** Sequences like `Alt + c` then `t` mean: press `Alt + c`, release, then press `t`.
- Use `/` to quickly jump to the search bar from anywhere in Drive.
- `Shift + Space` toggles selection without opening a file — useful for batch operations.
- `F2` to rename is one of the most useful shortcuts for file organization.
- `Ctrl + Alt + p` opens a quick preview panel without fully opening the file.
