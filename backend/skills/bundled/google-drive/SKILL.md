## Google Drive — Keyboard Shortcuts

Google Drive is the file management web app at drive.google.com.

> Press `Ctrl + /` (Windows) or `⌘ + /` (macOS) to display the keyboard shortcuts list inside Drive.

### Tips

- **Two-step shortcuts:** Sequences like `Alt + c` then `t` (Windows) or `Option + c` then `t` (macOS) mean: press the combo, release, then press the next key.
- Use `/` to quickly jump to the search bar from anywhere in Drive.
- `Shift + Space` toggles selection without opening a file — useful for batch operations.
- `F2` (Windows) or `Enter` (macOS) to rename is one of the most useful shortcuts for file organization.
- `Ctrl + Alt + p` / `⌘ + Option + p` opens a quick preview panel without fully opening the file.
- On macOS, `⌘` replaces `Ctrl`, `Option` replaces `Alt` for modifier-key shortcuts.
