## Google Chrome — Keyboard Shortcuts

Google Chrome is the primary browser for accessing Google Workspace apps.

---

### Tab and Window Shortcuts

| Action | Windows | macOS |
|---|---|---|
| Open a new window | `Ctrl + n` | `⌘ + n` |
| Open a new window in Incognito mode | `Ctrl + Shift + n` | `⌘ + Shift + n` |
| Open a new tab and jump to it | `Ctrl + t` | `⌘ + t` |
| Reopen previously closed tab | `Ctrl + Shift + t` | `⌘ + Shift + t` |
| Jump to next open tab | `Ctrl + Tab` or `Ctrl + Page Down` | `⌘ + Option + Right arrow` |
| Jump to previous open tab | `Ctrl + Shift + Tab` or `Ctrl + Page Up` | `⌘ + Option + Left arrow` |
| Jump to specific tab (1–8) | `Ctrl + 1` through `Ctrl + 8` | `⌘ + 1` through `⌘ + 8` |
| Jump to last tab | `Ctrl + 9` | `⌘ + 9` |
| Open previous page in tab history | `Alt + Left arrow` | `⌘ + Left arrow` or `⌘ + [` |
| Open next page in tab history | `Alt + Right arrow` | `⌘ + Right arrow` or `⌘ + ]` |
| Close current tab | `Ctrl + w` or `Ctrl + F4` | `⌘ + w` |
| Close current window | `Ctrl + Shift + w` or `Alt + F4` | `⌘ + Shift + w` |
| Minimize current window | `Win + Down arrow` | `⌘ + m` |
| Maximize current window | `Win + Up arrow` | (use green traffic light button) |
| Quit Chrome | `Alt + F4` (last window) | `⌘ + q` |
| Move tab to a new window | Drag tab out of tab bar | Drag tab out of tab bar |

### Mouse Shortcuts

| Action | Windows | macOS |
|---|---|---|
| Open link in new background tab | `Ctrl + click` | `⌘ + click` |
| Open link in new window | `Shift + click` | `Shift + click` |
| Open link in same tab | Drag link to address bar | Drag link to address bar |
| Open link in new tab | Drag link to tab bar | Drag link to tab bar |
| Close tab | Middle-click on tab | Middle-click on tab |
| Reopen closed tab | Middle-click on tab bar | Middle-click on tab bar |
| Zoom in | `Ctrl + scroll up` | `⌘ + scroll up` |
| Zoom out | `Ctrl + scroll down` | `⌘ + scroll down` |
| Scroll left/right | `Shift + scroll` | `Shift + scroll` |
| Back | `Alt + click back arrow` | `⌘ + click back arrow` |
| Forward | `Alt + click forward arrow` | `⌘ + click forward arrow` |
| Switch to specific tab | Click tab | Click tab |

### Tips

- **`Ctrl + l`** (Windows) or **`⌘ + l`** (macOS) is the fastest way to jump to the address bar and start typing a URL or search.
- **`Ctrl + Shift + t`** / **`⌘ + Shift + t`** reopens the last closed tab — invaluable for accidental closes. Works multiple times.
- Use **`Ctrl + 1–8`** / **`⌘ + 1–8`** to jump directly to a specific tab by position.
- **`F12`** / **`⌘ + Option + j`** opens DevTools — essential for web debugging.
- **`Ctrl + Shift + Delete`** / **`⌘ + Shift + Delete`** quickly opens the clear browsing data dialog.
- **Middle-click** on a link opens it in a new tab without switching to it.
- On macOS, `⌘` replaces `Ctrl`, `Option` replaces `Alt`, and `Fn + Delete` is forward delete.
