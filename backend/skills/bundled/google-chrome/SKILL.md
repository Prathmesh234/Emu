## Google Chrome — Keyboard Shortcuts

Google Chrome is the primary browser for accessing Google Workspace apps.

---

### Tab and Window Shortcuts

| Action | Shortcut |
|---|---|
| Open a new window | `Ctrl + n` |
| Open a new window in Incognito mode | `Ctrl + Shift + n` |
| Open a new tab and jump to it | `Ctrl + t` |
| Reopen previously closed tab | `Ctrl + Shift + t` |
| Jump to next open tab | `Ctrl + Tab` or `Ctrl + Page Down` |
| Jump to previous open tab | `Ctrl + Shift + Tab` or `Ctrl + Page Up` |
| Jump to specific tab (1–8) | `Ctrl + 1` through `Ctrl + 8` |
| Jump to last tab | `Ctrl + 9` |
| Open previous page in tab history | `Alt + Left arrow` |
| Open next page in tab history | `Alt + Right arrow` |
| Close current tab | `Ctrl + w` or `Ctrl + F4` |
| Close current window | `Ctrl + Shift + w` or `Alt + F4` |
| Minimize current window | `Win + Down arrow` |
| Maximize current window | `Win + Up arrow` |
| Quit Chrome | `Alt + F4` (last window) |
| Move tab to a new window | Drag tab out of tab bar |

### Tips

- **`Ctrl + l`** is the fastest way to jump to the address bar and start typing a URL or search.
- **`Ctrl + Shift + t`** reopens the last closed tab — invaluable for accidental closes. Works multiple times.
- Use **`Ctrl + 1–8`** to jump directly to a specific tab by position.
- **`F12`** opens DevTools — essential for web debugging.
- **`Ctrl + Shift + Delete`** quickly opens the clear browsing data dialog.
- **Middle-click** on a link opens it in a new tab without switching to it.
