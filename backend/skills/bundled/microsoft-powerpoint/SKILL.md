## Microsoft PowerPoint ‚Äî Keyboard Shortcuts (macOS)

Microsoft PowerPoint is a presentation application. On macOS it runs as a native desktop app. The interface has a ribbon toolbar at the top with tab groups (Home, Insert, Design, Transitions, Animations, Slide Show, Review, View). The left panel shows slide thumbnails for navigation. The center canvas is the main editing area. A notes pane is available at the bottom.
