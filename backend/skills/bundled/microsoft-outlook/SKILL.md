## Microsoft Outlook — Keyboard Shortcuts (macOS)

Microsoft Outlook is an email and calendar application. On macOS it runs as a native desktop app. The interface has a left sidebar showing mail folders (Inbox, Drafts, Sent, Archive, etc.). The center pane lists messages. The right pane (or bottom pane) shows the reading preview. The bottom navigation bar switches between Mail, Calendar, People, and Favorites views. A compose window opens in a new overlay or separate window.

### Tips

- **Quick view switching:** `⌘ + 1` (Mail), `⌘ + 2` (Calendar), `⌘ + 3` (People) instantly jumps between views.
- **Reading pane:** Toggle with `⌘ + \` (right) or `⌘ + Shift + \` (bottom) based on your workflow.
- **Flag shortcuts** (`Control + 1–6`) work from the message list — no need to open the message.
- **Archive:** `Control + E` archives the selected message immediately.
- **Search scoping:** `Option + ⌘ + F` searches the current folder; `⌘ + Shift + F` opens advanced search across all folders.
