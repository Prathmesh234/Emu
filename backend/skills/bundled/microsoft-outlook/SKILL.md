## Microsoft Outlook ŌĆö Keyboard Shortcuts (macOS)

Microsoft Outlook is an email and calendar application. On macOS it runs as a native desktop app. The interface has a left sidebar showing mail folders (Inbox, Drafts, Sent, Archive, etc.). The center pane lists messages. The right pane (or bottom pane) shows the reading preview. The bottom navigation bar switches between Mail, Calendar, People, and Favorites views. A compose window opens in a new overlay or separate window.
